# 🚀 Toolverse

**Toolverse** is a student-focused platform designed to help beginners choose the right tech domain with clarity and confidence.
It transforms confusion into a structured learning path using guided roadmaps, market insights, and curated resources.

---

## 🌟 Problem

Many students struggle to decide their career path in tech.
They spend hours watching YouTube videos, reading articles, browsing Reddit and LinkedIn—but still feel confused.

---

## 💡 Solution

Toolverse solves this by providing:

* 🎯 Clear domain guidance (Web Dev, Cybersecurity, Networking, etc.)
* 🗺️ Structured learning roadmaps
* 📊 Market trends & demand insights
* ⚙️ Relevant technologies for each domain
* 💼 Job roles and career opportunities
* 🎓 Curated courses (free & paid)

---

## ✨ Features

* 🔥 Interactive onboarding flow
* 🎨 Clean and modern UI (dark theme)
* 📚 Domain-specific roadmap pages
* 🧠 Beginner-friendly explanations
* 🔗 Clickable learning resources
* 🚀 Smooth animations and transitions

## ⚙️ Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/toolverse.git
cd toolverse
```

### 2️⃣ Install dependencies

**Frontend:**

```
cd frontend
npm install
npm start
```

**Backend:**

```
cd backend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the frontend:

```
REACT_APP_BACKEND_URL=http://localhost:5000
```
